# Teknoloji-Ma-azas-Stok-Takip-Otomasyonu
Mağazada bulunan ürünlerin giriş-çıkış işlemlerini ve stok durumlarını takip etmek için geliştirilmiş bir sistemdir. Bu projede ürün bilgileri, stok miktarları ve tedarikçi verileri SQL veri tabanı kullanılarak yönetilecektir. Sistem, stok azaldığında uyarı vererek mağaza yönetiminin ürün takibini kolaylaştırmayı amaçlamaktadır.
